## Overview

KilimoSTAT is an open data and statistics platform from the Ministry of Agriculture and Livestock Development in Kenya. This repository contains the React frontend application that provides interactive dashboards, data visualizations, and access to agricultural statistics.

## Features

- 📊 **Interactive Dashboards** - Visualize agricultural data with dynamic charts
- 🗺️ **Geospatial Analysis** - County-level mapping and analysis
- 📈 **Advanced Analytics** - Trend analysis and predictive insights
- 🔓 **Open Data Access** - Free access to agricultural datasets
- 📱 **Responsive Design** - Works on desktop, tablet, and mobile
- 🚀 **Fast Performance** - Optimized for quick loading

## Tech Stack

- **React 18** - UI framework
- **React Router** - Navigation and routing
- **Axios** - HTTP client for API calls
- **Chart.js** - Data visualizations
- **Tailwind CSS** - Styling and responsive design
- **Jest** - Testing framework

## Prerequisites

- Node.js 18.x or higher
- npm 9.x or higher
